


# DynFMI - Dynamic FM-Index for a Collection of Texts


Author: Wolfgang Gerlach


Compile library:

```bash
cmake .
make
```

creates: "libdynfmi.a"



if you want to change parameters: 

```bash
cmake .
ccmake .
make
```
